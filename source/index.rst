====================
Sponge Documentation
====================

Introduction
============

Welcome to SpongeDocs, the official documentation for the open-source `Sponge <http://spongepowered.org>`__ project.

The Sponge project currently has three main components:

* Sponge (coremod)
* Sponge API
* SpongeDocs

The articles within this documentation are intended to help developers and server owners become familiar with what Sponge has to offer. Common issues are addressed in the :doc:`faq`.

.. warning::

    Sponge is still in development. A download will not be available until later in 2015.
