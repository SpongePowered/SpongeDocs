=================
Installing an IDE
=================

.. note::

    An IDE (integrated development environment) is recommended for developing plugins with the Sponge API. An IDE auto-completes names and notifies you of errors, such as mismatched braces, in your code. We recommend using IntelliJ IDEA, though the choice is yours.
